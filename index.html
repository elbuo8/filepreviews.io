FilePreviews.io
Generate image previews and metadata from almost any kind of file

Interactive demo

FilePreviews.io is a service that generates image previews and metadata (on the cloud of course) for a lot of file types. Files uploaded to our service go through a pipeline that extracts as much data as possible and generates an image preview whenever possible. Some of the cool things we do are OCR, Adobe Photoshop (PSD) layer extraction, EXIF data extraction, Microsoft Office previews, Adobe Illustrator (AI) previews, PDF previews and a lot more.

We have 2 distinct APIs for interacting with the serive. The one we are using for this demo doesn't require a server on the API consumer's side. That's why we can host this page on GitHub. The other API is a typical REST service with a WebHook to notify you systems when your previews and metadata are ready.

While you are here check out our API libraries currently available for client side JavaScript, node.js and Python.
